# Blog API Assignment

This project is a simple RESTful Blog API built with Node.js, Express, and MongoDB to allow users to create, read, update and delete blog posts.

## Features Implemented
- CRUD Routes (POST, GET all, GET one, PUT, DELETE)
- Joi Validation on POST and PUT
- Thunder Client requests prepared for testing.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Cloudinary(for image uploads)
