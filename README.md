# Blog API Assignment

This project is a simple Blog API built with Node.js, Express, and MongoDB. 

## Features Implemented
- Article Schema
- CRUD Routes (POST, GET all, GET one, PUT, DELETE)
- Joi Validation on POST and PUT
- Thunder Client requests prepared for testing (7+ calls)
- Bonus: Search route implemented

## Notes
- MongoDB Atlas connection failed due to configuration issues during submission.
- Server runs successfully on port 5000.
- API routes are implemented and ready to work once MongoDB connection is fixed.
